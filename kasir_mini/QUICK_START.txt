📌 LANGKAH INSTALASI (5 MENIT!)

1️⃣  PERSIAPAN
   ✓ Download & Install XAMPP
   ✓ Jalankan XAMPP Control Panel
   ✓ Start Apache & MySQL

2️⃣  EKSTRAK FILE
   ✓ Ekstrak folder 'sistem_kasir' ke:
     C:\xampp\htdocs\sistem_kasir

3️⃣  IMPORT DATABASE
   ✓ Buka browser → http://localhost/phpmyadmin
   ✓ Buat database baru: kasir_minimarket
   ✓ Klik tab "Import"
   ✓ Pilih file: database.sql
   ✓ Klik "Go"

4️⃣  JALANKAN APLIKASI
   ✓ Buka browser
   ✓ Ketik: http://localhost/sistem_kasir
   ✓ Login dengan:
     Username: admin
     Password: admin123

5️⃣  SELESAI! 🎉
